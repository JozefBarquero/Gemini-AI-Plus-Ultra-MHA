### Go Beyond! Plus Ultra! 🚀

This is a simple and fun cosmetic script for fans of **My Hero Academia (Boku no Hero Academia)**.

#### What does it do?
It visually changes the text of your subscription in the Google Gemini interface from **"Plus"** to **"Plus Ultra"**. 

#### Features:
* **Lightweight & Fast:** It won't affect page performance.
* **Dynamic:** It uses a `MutationObserver` to ensure the text updates even when the page reloads or refreshes content dynamically via AJAX.
* **Safe:** It respects text fields (`textarea`, `input`) so it never alters what you type into the chat.
